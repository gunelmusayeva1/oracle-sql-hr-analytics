<h2>📌 About This Project</h2>

<p>In this project, I wrote <b>19 SQL queries</b> using Oracle's <b>HR Schema</b> database.
The queries solve real-life business problems — such as salary analysis, department statistics, employee rankings, and date calculations.</p>

<p><b>Database:</b> Oracle HR Schema<br>
<b>Tables used:</b> <code>employees</code>, <code>departments</code>, <code>locations</code>, <code>countries</code>, <code>regions</code>, <code>jobs</code></p>
